## Steps to Contribute

* ### *Fork the Repository*
* ### *Clone the repository*
    ```
    git clone https://github.com/your-username/repo-name.git
    ```
* ### *Create a new branch*
    ```
    git checkout -b your-branch-name
    ```
* ### *Add you changes*
    ```
    git add .
    git commit -m "Comments"
    ```
* ### *Push the changes*
    ```
    git push --set-upstream origin your-branch-name
    ```
